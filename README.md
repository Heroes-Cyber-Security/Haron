# Haron

DX-first infrastructure for blockchain CTF challenges.

## Developer Features
- Python as challenge runtime
- Foundry-native
- Memory-efficient
- Modular microservices
- CTFd auth integration (by Access Token)
- Telegram notification
- Dynamic flag by default, static flag by choice

## Challenge Capabilities
- Only EVM (for now)
- Normal single-chain vulns
- Cross-chain challenges
- MEV attack/strategy challenges
